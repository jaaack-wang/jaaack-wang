Measuring Iterative Temporal Reasoning with Time Puzzles

Zhengxiang Wang, Zeyu Dong

Preprint (2026)

Paper Code BibTeX

Abstract

Time Puzzles is a benchmark for measuring iterative temporal reasoning in language models. The benchmark builds controlled puzzle tasks that require step-by-step updates over time, so models must maintain and revise temporal states rather than rely on one-shot pattern matching. Results show that even strong recent models have clear gaps on these iterative settings, highlighting temporal consistency as an open challenge.

Main Results

Conclusion

The paper introduces Time Puzzles as a targeted testbed for iterative temporal reasoning and shows that current models still struggle when temporal updates must be chained over multiple steps. The results suggest that improving temporal state tracking and iterative reasoning should be a core direction for future model and evaluation design.
